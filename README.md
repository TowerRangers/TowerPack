# Tower Pack

기능은 순차적으로 추가 예정
<br/><br/><br/>

## ✨ 주요 기능

- ⚙ CVar 설정 관리
- 📐 특성 & 마법책 스케일 조절
- 🔔 LFG 파티 신청 알림

## 📌 추가 정보

- 모든 설정의 기본값은 비활성화 상태입니다.
- 프로필은 계정공유 프로필입니다. 한캐릭터에서 변경하면 다른캐릭터도 적용됩니다.
<br/><br/>
## ⚙ CVar 설정 관리

Plater 또는 Platynator 와 같은 네임플레이터 사용 시 CVar 값이 덮어씌워지는 문제를 자동으로 재정의합니다.

AdvancedInterfaceOptions 또는 CVar 명령어로 강제로 변경한 값도 유지됩니다.

- 순정 UI 환경에서 이름표 관련 옵션 두 항목을 모두 활성화하면,
아군 이름표가 체력바 없이 직업 색상으로 표시됩니다.
<br/><br/>
## 📐 특성 & 마법책 스케일 조절

기본 특성창 및 마법책 크기를 원하는 비율로 조정합니다.

Talent Tree Tweaks 또는 BlizzMove에서도 조절이 가능하지만,
불필요한 기능을 제외한 단순 스케일 조절용으로 제작되었습니다.

- 비율 조절 가능
<br/><br/>
## 🔔 LFG 파티 신청 알림

파티 구인 중 신청자가 있을 경우 알림을 표시하고
자동으로 파티 구성 창을 엽니다.

- 전투 중에는 알림만 표시됩니다.
- 파티 구성 창이 이미 열려 있는 경우 작동하지 않습니다.

설정 가능 항목

- 알림 메시지 변경
- Y축 위치 조절
- 폰트 크기 변경
- 표시 시간 및 페이드 애니메이션 설정
- 사운드 기능
- 테스트 기능
<br/><br/>
## 📦 전쟁 보급품 알림

전쟁 모드에서 보급상자가 도착하면 화면에 알림을 표시합니다.

귓속말 알림을 활성화하면,
비전투 상태에서 백그라운드 실행 중에도 작업 표시줄이 점멸합니다.
(우클릭 시 닫기 가능)

지원 확장팩

- 격전의 아제로스
- 어둠땅
- 용군단
- 내부 전쟁

설정 가능 항목

- X / Y 위치 조절
- 표시 시간 변경
- 귓속말 알림
- 사운드 기능
- 테스트 기능
<br/><br/>
## 💎 증강룬 사용 알림

우선순위에 따라 자동으로 감지합니다.

- 1순위 : 실체없는 증강의 룬
- 2순위 : 꿈결속 증강의 룬

가방에 해당 아이템을 보유 중일 때만 표시됩니다.
아이템이 없는 저레벨 캐릭터에서는 작동하지 않습니다.

설정 가능 항목

- Y축 위치 조절
- 아이콘 크기 변경
- 폰트 크기 변경
- 탈것 탑승 시 숨김 기능
<br/><br/>
## 🎁 이벤트 상자 자동 오픈

아래 축제 이벤트 던전 완료 보상 상자를 자동으로 개봉합니다.

- 온누리에 사랑을
- 가을 축제
- 할로윈 축제

전투 중 획득한 아이템은 전투 종료 후 자동으로 개봉됩니다.
여러 클라이언트를 동시에 실행하는 경우 특히 유용합니다.
<br/><br/>
## 👋 파티 / 공격대 자동 인사

내가 파티/공격대에 참가하거나
다른 플레이어가 내 파티에 참여할 경우 환영 메시지를 전송합니다.

- 내 공격대에 다른 플레이어가 참가할 경우에는 메시지를 전송하지 않습니다
- 설정한 딜레이 후 자동 전송

설정 가능 항목

- 딜레이 시간 조절
- 메시지 변경
